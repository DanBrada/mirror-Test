# Ping madness
if enabled on `&everyone` sends 50 `@everyone` pings to the discord channel message was sent to

## How to run
1) `npm install`
2) create `.env` file, that contains `CLIENT_SECRET` variable with value of your app client secret
3) `npm run test`
4) enjoy I guess
